# sandpit

Sandpit is gonna try to be a skeleton demo showing building a system with the following stack:

- Frontend
    - React
    - Bootstrap
    - TypeScript
- Data model
    - Hasura for the API
    - Postgres for the database
- Scheduled tasks
    - NATS for the task queue
    - Dkron for the task scheduler
    - Go for task execution
- Messaging
    - Redis
    - Centrifugo
